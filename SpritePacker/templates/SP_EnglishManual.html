『Sprite Packer』 is an app with which you can load images from Google Drive to pack them and download it as a sprite sheet and a sprite map.


※ Usage
  1. Download and install Google Drive program. Google Drive Download

  2. Create a folder in Google Drive.

  3. Upload images which you want to pack.

  4. Click the Google sign in button and choose an account to connect to Google


  5. Allow Google Drive readonly permission if you agree.

  6. Click the folder selection button.


  7. Select a folder in which images are located and click the 'Select' button.


  8. Select images to be packed and click '▶' button.


  9. Click the 'Pack' button.


  10. Write down sprite sheet name and click the download button below to download a packed sprite sheet image.

    If it doesn't work, do it in Chrome.

  11. Write down sprite map name and choose type and then click the download button below to download a sprite map file.


  12. If you want to save current sprites, write down atlas name and click save.

    You can load the atlas later on to keep working with.
